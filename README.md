Project Scope

This project demonstrates a DevSecOps Secure CI/CD Pipeline that automates software development, security testing, and deployment. The pipeline integrates security into every stage of the Software Development Life Cycle (SDLC), ensuring secure, reliable, and high-quality application delivery.

---

Project Workflow

1. Developer pushes code to GitHub.
2. GitHub Actions automatically triggers the pipeline.
3. Source code is downloaded.
4. Required dependencies are installed.
5. Static code analysis is performed.
6. Dependency vulnerability scanning is executed.
7. Unit tests are run.
8. Docker image is built.
9. Container image is scanned for vulnerabilities.
10. Application is deployed after successful validation.
11. Security and build reports are generated.

---

How to Run

1. Install Required Libraries

pip install bandit pytest

2. Clone the Repository

git clone https://github.com/yourusername/devsecops-pipeline.git

3. Navigate to the Project Folder

cd devsecops-pipeline

4. Run the Application

python app.py

5. Execute the CI/CD Pipeline

Push your code to the main or develop branch to automatically start the GitHub Actions workflow.
